# Publication Notes

## Included

- Canonical released dataset under `data/`
- Format conversion and baseline scripts under `code/`
- BRAT configuration and entity schema files under `metadata/`
- Example annotation figures under `docs/figures/`

## Excluded

The following repository content was intentionally left out of this staged public package:

- trained model weights and checkpoints
- experiment logs
- cached model outputs and temporary debug folders
- bulk internal annotation outputs
- backup and intermediate data directories
- API-specific and machine-specific experiment runner scripts

## Remaining Pre-Publication Tasks

- choose and add a real license
- add citation metadata
- decide whether to publish model checkpoints separately
- optionally package as a zip or repository release artifact
