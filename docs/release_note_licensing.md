# Release Note: Licensing and Redistribution

PrionNER uses a split licensing policy so that code, project-authored metadata, and third-party source text are not treated as if they had identical rights status.

- Code in `code/` is released under the MIT License.
- Project-authored documentation, schema files, metadata, annotations, and derived structured outputs are intended to be released under `CC BY 4.0`, to the extent the maintainers hold the necessary rights in those materials.
- Underlying article titles, abstracts, and other source text will remain in the public repository, but they are not blanket-relicensed by the project. Users should review `THIRD_PARTY_RIGHTS.md` before redistributing the source text.

For the legally safest downstream use, prefer the annotation layer, schema files, statistics, and project-authored metadata over redistribution of the underlying text itself.
